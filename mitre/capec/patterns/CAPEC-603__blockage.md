# CAPEC-603 - Blockage

**Abstraction:** Standard  -  **Likelihood:** Medium  -  **Severity:** High  -  **CAPEC:** [CAPEC-603](https://capec.mitre.org/data/definitions/603.html)

## Description
An adversary blocks the delivery of an important system resource causing the system to fail or stop working.

## Prerequisites
- This attack pattern requires knowledge of where important system resources are logically located as well as how they operate.

## Consequences
- **Availability**: Other (Blocking a resource from functional operation denies its availability to authorized users.)
