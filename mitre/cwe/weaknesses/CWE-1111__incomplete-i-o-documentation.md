# CWE-1111 - Incomplete I/O Documentation

**Abstraction:** Base  -  **Status:** Incomplete  -  **CWE:** [CWE-1111](https://cwe.mitre.org/data/definitions/1111.html)

## Description
The product's documentation does not adequately define inputs, outputs, or system/software interfaces.

Not Language-Specific

- **Other**: Reduce Maintainability
- **Other**: Increase Analytical Complexity
