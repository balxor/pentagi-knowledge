# T1059.007 - JavaScript (tooling)

Related ATT&CK context: JavaScript execution examples can overlap with
obfuscation, ingress tool transfer, drive-by compromise, HTA execution, and COM
automation.

## Node.js

```bash
# Execute an inline JavaScript expression
node -e "console.log(process.platform); console.log(process.version)"

# Execute a script file
node ./script.js

# Fetch text content using built-in HTTPS APIs
node -e "require('https').get('https://<host>/status.txt', r => r.pipe(process.stdout))"
```

## Windows Script Host JScript

```batch
:: Console-based JScript execution
cscript.exe //nologo script.js

:: GUI-based JScript execution
wscript.exe script.js
```

```js
// script.js - run a local command through Windows Script Host
var shell = new ActiveXObject("WScript.Shell");
shell.Run("cmd /c whoami", 0, true);
```

## HTML Application JScript

```html
<!-- Execute with: mshta.exe demo.hta -->
<html>
<head>
<script language="JScript">
var shell = new ActiveXObject("WScript.Shell");
shell.Run("cmd /c whoami", 0, true);
window.close();
</script>
</head>
<body></body>
</html>
```

## macOS JavaScript for Automation

```bash
# Run inline JXA with osascript
osascript -l JavaScript -e 'Application.currentApplication().includeStandardAdditions = true; console.log("jxa")'

# Run a JXA file
osascript -l JavaScript ./script.js
```

```js
// script.js - run a local command through JXA
ObjC.import("stdlib");
$.system("whoami");
```
